# 📊 Student Digital Behavior & Productivity Intelligence System

## 📌 Project Overview

The **Student Digital Behavior & Productivity Intelligence System**
is an end-to-end Data Analytics & Machine Learning project designed to analyze how digital behavior, lifestyle habits, and wellness patterns impact student productivity and academic performance.

This project simulates a real-world behavioral analytics system similar to those used in:
- EdTech platforms
- Wellness analytics
- Product analytics
- User behavior intelligence systems

The project combines:
- Data Analytics
- Behavioral Analytics
- Business Intelligence
- Machine Learning
- Dashboard Storytelling
- Interactive App Development

---

# 🎯 Business Problem

Educational institutions often struggle to identify:
- students at burnout risk,
- declining productivity patterns,
- unhealthy digital habits,
- performance-impacting behaviors.

This project aims to answer questions such as:

- What behaviors affect academic performance?
- Does sleep influence productivity?
- Which student groups are most at risk?
- Can burnout risk be predicted?
- What patterns correlate with high performance?
